   T:\VestigantSpotlightInv_V0_9_59\build-msvc\Release\VestigantSpotlight.exe

   Q:\SpotlightCase\TestiOS_WhatsApp_V0_9_48_ReusedCache\VestigantSpotlight.case.sqlite

5. Select a wide result view such as:

   - iOS - Spotlight Text Context Review
   - iOS - Missing From FFS Text Detail
   - iOS - Bplist/NSKeyedArchiver Detail

7. Confirm the bottom "Selected Row Metadata / All Fields" pane updates for each selected row and shows fields vertically without horizontal grid scrolling.
8. Confirm copying text from the details pane works.

No reuse-cache or fresh-ZIP ingest is required for this GUI-only feature test.
