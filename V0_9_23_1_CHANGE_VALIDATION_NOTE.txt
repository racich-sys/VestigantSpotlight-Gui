# Vestigant Spotlight v0.9.24 Change / Validation Note

## Purpose

V0_9_24 is a compile-fix release for the Windows/MSVC build failure in V0_9_23:

- `src\db\case_db.cpp(1970): error C2026: string too big, trailing characters truncated`

## Change

- Refactored oversized SQL raw string-literal sequences in `src/db/case_db.cpp` into a runtime `joinSql(...)` helper fed by smaller raw string segments.
- Preserved the SQL content and V0_9_23 investigator behavior.
- Renamed scripts and version metadata to V0_9_24 / 0.9.24.

## Validation

- Linux CMake configured successfully.
- Linux compilation reached and passed `src/db/case_db.cpp`, confirming the refactored source is syntactically valid in this environment.
- The local Linux build timed out later during the larger build, so full Windows/MSVC validation is still required.
