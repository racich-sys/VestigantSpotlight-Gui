# Vestigant Spotlight v0.9.24 Change / Validation Note

## Purpose

V0_9_24 continues from the successful V0_9_23_1 run. The run completed without DB/WAL growth or export stall, but the review showed intentionally even/sampled counts and compact persistence behavior that needed to be explicit for investigator validation.

## Changes

- Added `exports/parser_limits_and_suppression_summary.csv` to every export package.
  - Records native record/block limits.
  - Records whether full native DB, full FFS inventory, and app DB record materialization were enabled.
  - Explains that normal iOS mode keeps compact raw key/value/date persistence.
  - Reports raw key/value/date ratios and text-context row counts.
  - Explains thin-upload sampling and override options.
- Persisted run options into `case_info`, including export profile, skip-hash mode, diagnostic full-native DB mode, FFS/app DB materialization flags, native record/block limits, and DB guardrail bytes.
- Added `vw_ios_spotlight_text_context_review` for GUI/local SQLite review of same-record Spotlight text context.
- Added GUI view: `iOS - Spotlight Text Context Review`.
- Added normal export: `ios_spotlight_text_context_review_sample.csv`.
- Added upload-sample export for the same text-context review surface.
- Updated `CASE_REVIEW_SUMMARY.txt` to include explicit run-limits/suppression status and ratios so even row counts are explainable.
- Updated upload/readme/review index references to point users to the limits report.

## Validation status

- Linux build/self-test should be run after patching.
- Windows/MSVC build remains the controlling validation because prior compile failures were MSVC-specific oversized literal issues.

## Expected review impact

V0_9_24 should make it clear whether apparently even counts are full parse counts, deliberate compact persistence, or thin-upload samples. It also gives investigators a direct GUI surface for the actual recovered Spotlight text context that was previously visible mainly through Missing From FFS rows or broad string-probe samples.
